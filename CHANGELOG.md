# Changelog

## 0.1.0 - Initial release

CodeSlot puts one unobtrusive sponsored slot in your VS Code status bar and pays
you AI usage credits while you code.

- **One sponsored slot** in the status bar, with a live earnings readout.
- **Earn while you code** - credits accrue on focused impressions (~5s dwell)
  and on click-throughs. CodeSlot reads only whether the window is focused; it
  never touches your code, files, or projects.
- **Sign in with GitHub** to earn (keeps rewards honest - no anonymous farming).
- **Wallet view** - balance, today's earnings, and recent activity, shown as
  **real money** ($ or ₹) so you always know what your credits are worth.
- **Redeem credits** for a real OpenRouter API key (Claude, GPT, Gemini, and
  more), loaded with the redeemed value and shown once to copy.
- **Privacy by design** - an anonymous device id only, your OpenRouter key kept
  in your OS keychain, and traffic only to a fixed https backend.
- Commands: Open Wallet, Redeem Credits, Show Balance, Pause / Resume Ads, Open
  Current Ad, Sign in / out, and Delete My Data.
